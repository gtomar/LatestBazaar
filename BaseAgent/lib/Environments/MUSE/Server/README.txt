Need Java & MySQL to run the server

1. Create palaver database in MySQL

2. Run MUSE_v2.0.jar from Deployment folder to launch the server
Command Line:
java -jar MUSE_v2.0.jar

3. Test that server is running and administer chat rooms using Admin folder

4. Double click on ServerAdmin_v2.0.0.jar to launch admin

5. Enter hostname or IP address of the machine running the server (Localhost: 127.0.0.1)
Port: 9000
Username: admin
Password: See in serverConfig.xml in deployment folder

6. Double click on PalaverDesktop.jar in ..\Client folder to run clients
